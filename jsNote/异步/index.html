异步模式：

  1. 回调函数
  2. 事件监听
  3. 观察者模式
  4. promise对象

回调函数

    var f1 = function (callback) {
        setTimeout(function () {
            // f1任务代码
            callback();
        }, 1000);
    }
    var f2 = function () {
        alert("f2");
    }

    f1(f2);

采用这种方式，我们把同步操作变成了异步操作，f1不会堵塞程序运行，相当于先执行程序的主要逻辑，将耗时的操作推迟执行。
