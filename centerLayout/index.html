居中1 —> margin: 0 atuo 
/********** 居中块1 *********/
.center1 {
    margin: 10px auto;
}

具体来说就是把要居中的元素的margin-left和margin-right都设为auto，此方法只能进行水平的居中，且对浮动元素或绝对定位元素无效。

center1#center1_1 {
    position: absolute;
}
                    
.center1#center1_2 {
    float: left;
}
居中2 —> text-align:center 
/********** 居中块2 *********/
.center2-out {
    width: 800px;
    height: 200px;
    border: 1px solid #000;
    text-align: center;
}
                    
.center2-in {
    display: inline-block;
}

只能对图片，按钮，文字等行内元素(display为inline或inline-block等)进行水平居中。但要说明的是在IE6、7这两个奇葩的浏览器中，它是能对任何元素进行水平居中的。

居中3 —> line-height 
/********** 居中块3 *********/
.center3 {
    width: 800px;
}
.center3 p{
	line-height:150px;
}

把文字的line-height设为文字父容器的高度，适用于只有一行文字的情况。

居中4 —> 使用表格 
/********** 居中块4 *********/

.center4-table {
    width: 800px;
    height: 300px;
    border: 1px solid #000;
}

.center4-td {
    text-align: center;
    vertical-align: middle
}

.center4-td .center4-td-in {
    display: inline-block;
    width: 100px;
    height: 100px;
}

居中6 —> 使用绝对定位来进行居中 
/********** 居中块6 *********/

.center6-out {
    position: relative;
}

.center6-in {
    position: absolute;
    top: 50%;
    left: 50%;
    width: 100px;
    height: 100px;
    margin-top: -50px;
    margin-left: -50px;
}
居中7 —> 另一种使用绝对定位来居中的方法 
/********** 居中块7 *********/

.center7-out {
    position: relative;
}

.center7-in {
    position: absolute;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    width: 100px;
    height: 100px;
    margin: auto;
}
